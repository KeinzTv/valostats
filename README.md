# ValoStats is a website that allows you to view some Valorant stats.

## Run in development environment :
### Run the front :
``
npm install
``
####
``
npm run start
``

### Run the back :
``
cd api
``
####
``
npm install
``
####
``
npm run dev
``

## Run for prod :
See https://cli.vuejs.org/guide/deployment.html

### Run the front :
``
npm install
``
####
``
npm run build
``

### Run the back :
``
cd api
``
####
``
npm install
``
####
``
npm run start
``
